# Linked Business Web Test API

## Endpoints

### 1. Get Astronauts by Nationality
**Using Postman**

**URL:** `/api/astronauts`

**Method:** `POST`

**Body Parameters:**
- `nationality`: The nationality of the astronauts to be fetched.

**Response:**

[
    {
        "name": "Astronaut Name",
        "age": 35
    },
    ...
]
